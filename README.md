# Jimoh Abdulquyum Portfolio

A modern, responsive, and visually appealing portfolio built with **React** and **React-Bootstrap**. Showcases my projects, skills, and contact information, designed for both desktop and mobile users.

## Features

- **About Section**: Introduces me and my tech skills.
- **Projects Section**: Displays my recent projects with GitHub and demo links.
- **Contact Form**: Fully functional contact form using **EmailJS**.
- **Responsive Design**: Works across different devices and screen sizes.
- **Animations & Effects**: Smooth transitions and particle background for visual appeal.

## Technologies Used

- **Frontend**: React, React-Bootstrap, React Icons, CSS
- **Email Integration**: EmailJS
- **Version Control**: Git, GitHub
